How to use Sticky Note




Non-persistent route:

  1. Drag and drop the new sticky icon onto the Sticky Board.

  2. To open the sticky editor, double click on the newly created sticky.
  3. Once inside the editor, input the text that you wish and select a color for the sticky note.
  4. Click the "Save Note" button to save the sticky note and to return back to the sticky board.



Persistent route:

  1. Create a new user by typing in your desired username in the top of the window. Then click "Create".
  2. If you already have created an account, simply select your username from the dropdown menu. Your sticky board will be automatically populated.
  3. Drag and drop the new sticky icon onto the Sticky Board.

  4. To open the sticky editor, double click on the newly created sticky.
  5. Once inside the editor, input the text that you wish and select a color for the sticky note.
  6. Click the "Save Note" button to save the sticky note and to return back to the sticky board.
  7. Lastly, click the "Save Board" button on the left sidebar when you are done editing your board.
